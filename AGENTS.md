# AGENTS.md — Каркас проекта ItoCook

## 1. Стек и Архитектура
- **Frontend:** Nuxt 4 (Vue 3, Script Setup, TS) + Tailwind CSS + Nuxt UI
- **Backend/CMS:** Directus + PostgreSQL
- **Микросервис:** FastAPI (Python) — для сложных финансовых расчетов
- **Оркестрация:** Docker Compose (монорепозиторий)
- **Шрифт & Иконки:** Jost / Phosphor Icons (`@phosphor-icons/vue`)

## 2. Ссылки на подробные планы (Обязательно прочесть перед работой!)
- Подробный план разработки, фазы, MVP и ТЗ: `docs/plan-main.md`
- Логика работы, сценарии и промпты для AI-ассистента: `docs/plan-ai.md`
- Визуальные референсы и дизайн-концепт: папка `docs/design/` 

## 3. Правила игры
- Мы используем установленные локальные навыки из `~/.config/opencode/skills/`.
- Любые действия по генерации кода, структуры или контейнеров сначала утверждаются в режиме `/plan`.
